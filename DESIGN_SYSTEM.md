# Design System

## Typography

- Font Family: M PLUS 2（Google Fonts）
- Font Size: 全要素 10px 統一
- Weight: 見出し類 Regular (400) / 本文・説明文 Extra Light (200)

## Color

- Background: #F5F5F5
- Text: #333333
- Border: #E0E0E0

## Spacing

- セクション間余白: 120〜160px
- 罫線: 1px solid #E0E0E0

## Rules

- サイズのバリエーションを増やさない
- ウェイトの差で情報の階層を表現する
- 余白と罫線でリズムを作る
